  2) 정보저장소
    - :
    - :
      = 를 줄이는 전략(작동기억의 용량을 효과적으로 쓰는 법)
        + : 정보를 보다 크고 의미 있는 단위로 묶는 정신과정
        + : 무의식적으로 할 수 있는 정신적 조작을 수행하는 능력
        + : 시각, 언어, 청각정보 등을 분산하여 처리
    - : 무한한 정보를 영구적으로 저장할 수 있는 곳
  3) 인지처리과정
    - : 자극에 의식적으로 초점을 두는 과정
    - : 자극에 의미를 부여하기 위해 그것을 해석하는 방법
    - : 정보 유지를 위해 마음 속 또는 소리 내어 정보를 반복함
    - : 제시된 정보를 퍼리 가능한 형태로 변형하는 과정
      = : 기존에 가지고 있던 정보를 새로운 정보에 연결하여 정보를 유의미한 형태로 저장하는 과정(예시, 유츠 등)
      = : 관련 있는 내용을 공통 범주나 유형으로 묶는 과정
      = : 정보에 대한 시각적 이미지를 머릿속에 표상하는 과정. 에 따르면 학습자는 정보를 저장할 때, 언어와 이미지의 두 가지 방법을 사용하며, 이는 정보의 저장뿐만 아니라 인출 및 활용을 촉진함
    - : 장기기억 속에 있는 정보를 꺼내고자 탐색하는 과정
      = : 특정한 정보에 접근하는 것을 도와주는 자극. 인출에 실패한 기억도 이를 제시하면 살아날 수 있음

        + : 자신의 인지 상태의 변화 정도를 평가함

4. 문제해결과 학습 1. 학습의 전이 :
(1) 전이의 종류
  1) : 이전 상황의 학습이 다른 상황의 학습을 촉진
  2) : 이전 상황의 학습이 다른 상황의 학습, 수행을 방해
(2) 전이이론
  1) : 정신능력이 으로 되며, 이것이 광범위한 영역으로 확대된다는 주장 (교육과정, 주의)
  2) : 여러 학습과제 사이에 나 이 존재할 때만 전이가 일어난다고 보는 입장 (교육과정, 주의)
  3) : 선행학습에서 획득된 나 을 후속학습에 활용할 수 있을 때 전이가 일어남(교육과정)
  4) : 을 전이의 결정적인 요소로 보는 이론(학습자의 자발적인 우너리 발견). 선행학습 장면의 (구조적인 측면)에 대한 통찰이 발생해야 다른 장면으로 전이가 가능하다고 주장( 교육과정)
2. 문제해결 과정과 영향 요인
(1) 문제해결전략
  1) : 정해진 규칙과 절차에 따라 신속하고 정확하게 문제를 해결하는 방법(구조화된 문제 해결)
  2) : 문제해결을 위한 비규정적이고 직관적인 전략. 비구조화된 문제 또는 잘 정의되지 ㅇ낳은 문제는 어림법의 사용이 필요함
    - 수단-목표 분석 : 상위목표를 여러 단계의 하위목표로 세분화한 후, 각 하위목표를 달성할 수 있는 적절한 수단을 탐색하여 적용
    - 유추하기 : 문제상황을 기존에 경험한 적이 있는 보다 익숙하고 간편한 문제에 대응시켜 문제해결을 지원하는 전략. 개인이 가지고 있는 경험이나 배경지식이 중요
